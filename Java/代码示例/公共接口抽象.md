[优化]
反过来：将策略的方法定义成抽象，将组合的方法定义成公共
好处：策略的实现类比较多，组合的只要一个实现类

或者：
公共接口->父级实现->子级继承[重写方法]
在父级实现中 可以使用 策略获取子级
